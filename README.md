# Bakery — сайт современной пекарни

## Описание

Bakery — адаптивный сайт для пекарни, который презентабельно демонстрирует продукцию, фото, отзывы клиентов и контактные данные. Проект предназначен для малого бизнеса, желающего продвигать свою продукцию и получать новых клиентов онлайн.

## Функции

-Адаптивный дизайн для десктопа и мобильных устройств

-Галерея с интерактивным просмотром изображений

-Список продукции с карточками и описаниями

-Секция отзывов клиентов с плавной анимацией

-Интерактивная карта с адресом

-Контакты, режим работы, соц. сети (ВКонтакте, Instagram, WhatsApp)

-SEO-оптимизация: метатеги, alt для картинок, микроразметка Bakery (schema.org)

-Favicon, оформление, легкая редакция кода

## Технологии

-HTML5, CSS3 (Grid, Flexbox, адаптивные медиазапросы)

-JavaScript (минимальные скрипты для галереи)

-BasicLightbox для просмотра изображений

-Яндекс.Карты (iframe)
